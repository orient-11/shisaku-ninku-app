# 試作課 人工管理アプリ セットアップ手順

## 必要なもの
- Google アカウント
- Google スプレッドシート（新規作成）

---

## ステップ 1: スプレッドシートを作成する

1. [Google スプレッドシート](https://sheets.google.com) を開く
2. 「空白のスプレッドシート」を新規作成
3. スプレッドシートに名前をつける（例：`試作課 人工管理`）

---

## ステップ 2: Apps Script を設定する

1. スプレッドシートのメニューから **「拡張機能」→「Apps Script」** を開く
2. 左側のファイル一覧で「コード.gs」を選択し、**既存のコードをすべて削除**
3. `Code.gs` の内容をコピーして貼り付ける
4. 左上の「＋」ボタンから **「HTMLファイル」** を追加し、ファイル名を `Index` とする
5. `Index.html` の内容をコピーして貼り付ける
6. 上部の **「💾 保存」** ボタンをクリック

---

## ステップ 3: Web アプリとして公開する

1. 右上の **「デプロイ」→「新しいデプロイ」** をクリック
2. **「種類の選択」** で「ウェブアプリ」を選択
3. 設定：
   - **説明**: 試作課 人工管理（任意）
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: 社内の人全員（または「全員」）
4. **「デプロイ」** をクリック
5. 表示された **Web アプリの URL** をブックマーク・共有する

---

## ステップ 4: 初期セットアップ

1. Web アプリを開く
2. 右上の **「初期設定」** ボタンをクリック
   → スプレッドシートに以下のシートが自動作成されます：
   - `職人` — 職人マスター
   - `スケジュール` — 案件マスター
   - `作業ログ` — 作業記録（自動記録）
   - `計測中` — アクティブタイマー管理（自動管理）

---

## ステップ 5: 職人・案件を登録する

1. アプリの **「設定」タブ** を開く
2. 職人名を入力して「追加する」
3. 案件名・期間を入力して「追加する」

---

## 使い方

### 計測を始める
1. **「計測」タブ** で職人と案件を選択
2. **「▶ 計測スタート」** を押す → タイマーが動き始める

### 計測を終える
1. 計測中カードの **「⏹ 終了・記録」** を押す
2. 備考を入力して **「終了して記録」**
3. スプレッドシートの「作業ログ」に自動保存される

### 人工数を確認する
- **「集計」タブ** で期間・職人・案件を選んで「集計する」
- **1人工 = 8時間** で自動換算して表示

---

## スプレッドシートの列構成

### 作業ログ シート
| 列 | 内容 |
|----|------|
| ID | 自動生成ID |
| 日付 | 作業日（yyyy/MM/dd）|
| 職人名 | 担当職人 |
| 案件名 | スケジュール名 |
| 開始時刻 | HH:mm:ss |
| 終了時刻 | HH:mm:ss |
| 作業時間(分) | 経過分数 |
| 人工数 | 作業時間÷480分 |
| 備考 | 任意メモ |

---

## トラブルシューティング

| 症状 | 対処法 |
|------|--------|
| 「初期設定」ボタンが効かない | Apps Script の権限を承認してください |
| 職人・案件が表示されない | 「設定」タブで登録してください |
| タイマーが再起動後も残っている | スプレッドシートの「計測中」シートを手動で確認 |
| 時間がずれる | Apps Script のタイムゾーンを Asia/Tokyo に設定してください |
